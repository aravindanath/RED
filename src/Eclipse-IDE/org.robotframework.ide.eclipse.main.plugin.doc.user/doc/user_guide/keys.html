Useful RED key shortcuts

Quick Fix - CTRL+1

Quick Fix allows to perform predefined actions for common problems. It can be activated on item only when light bulb icon is present next to line number.

Code assistance - CTRL+SPACE

Provides Robot Framework test items assist. Cycle through modes with CTRL+SPACE. Working modes are customisable in Preferences.

Block comment - CTRL+?

Sets multi-line comments on the selected text in Source editor.

Automatic source formatting - CTRL+SHIFT+F

Apply simple source code reformat in Source editor.
